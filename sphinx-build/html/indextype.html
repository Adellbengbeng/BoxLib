IndexType¶

class IndexType¶

Located in Src/C_BaseLib/IndexType.H and Src/C_BaseLib/IndexType.cpp.

The class IndexType defines an index as being cell based or node (edge) based in each of the c:macro:BL_SPACEDIM directions. This class defines an enumerated type CellIndex to be either CELL or NODE; i.e. each of the c:macro:BL_SPACEDIM dimensions must be either CELL or NODE.
