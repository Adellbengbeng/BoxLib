class Orientation¶

Located in Src/C_BaseLib/Orientation.H and Src/C_BaseLib/Orientation.cpp.

This class encapsulates the orientation of the faces of a Box by providing an ordering of each of the faces of a Box in BL_SPACEDIM dimensions. This allows iterating over all the faces of a Box. The ordering first traverses the BL_SPACEDIM low sides from direction 0 .. BL_SPACEDIM-1 and then the BL_SPACEDIM high sides from direction 0 ..  BL_SPACEDIM-1. This class defines an enumerated type Side to be either low or high.

Example

for (OrientationIter face; face; ++face) {
  int iface = face();
  int idir = face().coordDir();
  std::cout << "Face index: " << iface
            << ", Dimension index " << idir << std::endl;
}
