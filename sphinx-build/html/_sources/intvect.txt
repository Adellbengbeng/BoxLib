IntVect
=======

.. cpp:class:: IntVect

  IntVect is an implementation of an integer vector in a
  SPACEDIM-dimensional space, where SPACEDIM is 1, or 2, or 3.  It
  represents a point in a discrete space.  IntVect values are accessed
  using the operator[] function.  In addition, the basic arithmetic
  operators have been overloaded to implement scaling and translation
  operations.
