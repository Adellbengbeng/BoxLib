Welcome to BoxLib Reference 
============================

*This version of the documentation is built using a combination of Sphinx 
as well as Doxygen to extract documentation embedded from the BoxLib source*

BoxLib is a mature software framework for building massively parallel
adaptive mesh refinement (AMR) applications.  BoxLib and a collection
of BoxLib application codes are public available at
https://github.com/BoxLib-Codes/.  BoxLib has C++ and Fortran
versions.  This reference manual documents the C++ version.
